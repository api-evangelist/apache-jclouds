# Apache jclouds (apache-jclouds)
Apache jclouds is an open-source multi-cloud toolkit for the Java platform that provides a portable abstraction for cloud APIs. It supports over 30 cloud providers including AWS, Azure, GCP, and OpenStack.

## APIs

### Apache jclouds
jclouds provides a unified Java and Clojure API for compute (VM management), blobstore (object storage), and provider-specific APIs across 30+ cloud providers including AWS, Azure, GCP, DigitalOcean, and OpenStack.
